# Контрольная работа — Lesson 6K

В рамках контрольной работы были разработаны автотесты
на Selenium WebDriver с использованием pytest.

## Выполненные задания

Создана отдельная ветка `lesson6K` и папка `06K_lesson`,
в которой реализованы три автотеста.
Во всех тестах используются явные ожидания `WebDriverWait`,
функция `sleep()` не применяется.

### 1. Форма (test_01_form.py)
- Открывается страница с формой
- Заполняются все поля, кроме Zip code
- Нажимается кнопка Submit
- Проверяется, что:
  - поле Zip code подсвечено красным
  - остальные поля подсвечены зелёным  
- Браузер: Edge

### 2. Калькулятор (test_02_calc.py)
- Открывается страница калькулятора
- В поле delay вводится значение 45
- Выполняется операция 7 + 8
- Проверяется, что результат 15 появляется через ожидание  
- Браузер: Chrome

### 3. Покупка (test_03_shop.py)
- Открывается сайт https://www.saucedemo.com/
- Выполняется авторизация пользователем standard_user
- В корзину добавляются три товара
- Выполняется оформление заказа
- Проверяется, что итоговая сумма равна $58.29  
- Браузер: Firefox

## Запуск тестов

Запуск всех тестов:
python -m pytest -q 06K_lesson
Для проверки стиля используется flake8:
flake8 06K_lesson
Ожидаемый результат — без ошибок и предупреждений.
